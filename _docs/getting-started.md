1. Create a new repository with a Jekyll `.gitignore`
2. Create a `_config.yml` with `repository`, `permalink`, `plugins` and `remote_theme`
3. Create a `index.md`
4. Go to **Settings > GitHub Pages** and set the Source as *master branch*
